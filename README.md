# Sunrise - Morning Experience App

Create and share personalized "Good Morning" experiences with beautiful animations and voice notes.

## Features

- 🎨 Beautiful themed experiences (Romantic, Calm, Playful)
- 🎙️ Voice note recording and playback
- ⏰ Schedule delivery time
- 🔗 Shareable unique links
- 📱 Mobile-first responsive design
- ✨ Smooth animations with Framer Motion

## Tech Stack

- **Frontend**: React, TypeScript, Vite, TailwindCSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB, Cloudinary

## Prerequisites

- Node.js (v16 or higher)
- MongoDB database (local or Atlas)
- Cloudinary account for audio storage
